# GTK-creator
GTK+ Creator - open-source (free software) IDE for Vala and GTK3

# Features:
1. Built-in Terminal (have some bugs)
2. Documentation Viewer
3. Code Editor (NOT FINISHED)
4. GIT Client

# TODO:

# Depends on:
* valac-0.30+
* gdk-3.0+
* vte-2.91+
* webkit2gtk-4.0+
* gio-2.0+
* gtksourceview-3.0+

# Make and install:
````bash
$ make
$ sudo make install
````
