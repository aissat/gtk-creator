# gtk-creator
Repository for GTK+ Creator - OpenSource IDE for Vala and GTK3

# Deps (of course, -dev versions (valac - compiler))

valac-0.30

gdk-3.0

vte-2.90

webkit2gtk-4.0

gio-2.0

gtksourceview-3.0

# Make and install
Just run this from linux
````bash
$ make
$ sudo make install
````

# Run
Just run `gtkcreator`
